# PARALLEL DRUMS - AUX (LOGIC - SD3)


## **Drums Section**

- **#1 - Parallel Punchy**
Chain with Saturn / Pro-Q4 / 1176 (ratio 4:1, att 3, rel 7), low cut 120 Hz → punch and saturation.

- **#2 - Ambient/Dark Parallel**
Multiband comp → dark EQ (cut 10-12 kHz, boost 400-500 Hz) → tape sat → reverb dark 2-3s.

- **#3 - Punch Electronic/Modern Parallel**
Transient Designer (Attack +, Sustain -) → 1176 All Buttons In → EQ 60-80 Hz + 3-4 kHz → Saturn Warm Tube type saturation.

- **#4 - Drums Bus Ambient/Dark**
Dark EQ (high cut, 200 Hz boost) → Light VCA/Opto Compressor (2-3 dB GR) → Dark Short Room (Space Designer, mix ~10%) → Light chorus optional.

- **#5 - Drums Bus Punchy**
EQ for presence → Comp VCA 2:1, medium attack, fast release → light saturation → limiter punch.
