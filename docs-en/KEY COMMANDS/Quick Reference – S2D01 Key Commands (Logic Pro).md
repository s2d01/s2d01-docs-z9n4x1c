# Quick Reference - S2D01 Key Commands (Logic Pro)

|**Tasto**|**Funzione**|**Uso pratico**|
|---|---|---|
|**F5**|Create New Aux Channel Strip|Add an FX bus now → Light → Full|
|**F6**|Bypass All Plug-ins (channel selected)|Take FX buses offline → Light mode|
|**F7**|Delete Selected Channel Strip|Clean the mixer from unused auxes|
|**F8**|Toggle Bypass Sends (assigned by you)|Turn on/off all sends of a track|

---

### Additional useful shortcuts

- **⌥Click** on Send knob → reset to -∞ (send OFF).  
    
- **Shift+⌥Drag** → copy Send level to multiple tracks.  
    
- **Ctrl+⌘+↑/↓** → quickly scroll between Input/Bus/Output in the Mixer.
