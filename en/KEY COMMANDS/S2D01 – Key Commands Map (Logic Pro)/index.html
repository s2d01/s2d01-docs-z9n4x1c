    S2D01 - Key Commands Map (Logic Pro)

    ASCII Keyboard Layout

    ┌────┬────┬────┬────┬────┬────┬────┬────┬────┬────┬────┬────┐
| F1 | F2 | F3 | F4 | F5 | F6 | F7 | F8 | F9 | F10 | F11| F12|
└────┴────┴────┴────┴────┴────┴────┴────┴────┴────┴────┴────┘
                       ↑ ↑ ↑ ↑
                       | | | |
                       | | | | └── F8 = Toggle Bypass Sends
                       | | | └────── F7 = Delete Channel Strip
                       | └─────────── F6 = Bypass All Plug-ins.
                       └──────────────── F5 = Create New Aux

    Compact Table

    Tasto Funzione Uso pratico
    F5 Create New Aux Channel Strip Add an FX bus (Light → Full)
    F6 Bypass All Plug-ins Take an Aux to Light mode (FX off)
    F7 Delete Selected Channel Strip Clear the mixer of unnecessary Auxes
    F8 Toggle Bypass Sends Turn on/off all sends on a track

    Useful Extras

    • ⌥Click on Send knob → reset to -∞ (send OFF).

    • Shift+⌥Drag → copy Send level to multiple tracks.

    • Ctrl+⌘+↑/↓ → scroll between Input / Bus / Output in the Mixer.

    -> With this map you have:

    • ASCII visual scheme (keyboard map).

    • Compact table (instant summary).

    • Extra tricks to speed up sends and mixers.
